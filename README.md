# 2D Shooting Game (Unity)

Unityで制作した2Dシューティングゲームです。  
プレイヤーを操作して敵を倒し、ステージクリアまたはゲームオーバーを目指します。
参考動画：https://www.youtube.com/watch?v=vyoV7SEig38&list=PLCImhINtRXjfnlAOlr8FAnAuqmxpJACtM

---

## 🎮 ゲーム概要

- 2D縦スクロールシューティング
- プレイヤーは移動と弾の発射が可能
- 敵を倒すとエフェクト発生
- HP制システムあり
- ゲームオーバー / ゲームクリア演出あり
- ボス撃破時に演出（時間減速・エフェクト）

---

## 🕹 操作方法

| 操作 | 内容 |
|------|------|
| 移動 | キーボード / スティック入力 |
| 攻撃 | 自動 or 入力トリガー（実装に依存） |

※ Input System を使用しています

---

## 🧩 主なシステム

### プレイヤー
- Rigidbody2Dによる移動制御
- HP管理
- 無敵時間（被ダメージ時の点滅）
- 弾の発射システム

### 敵
- プレイヤー追尾または直進移動
- 弾攻撃（派生可能構造）
- 被弾時ダメージ処理
- 死亡エフェクト生成

### 弾（Bullet）
- 直進移動
- 威力データ保持

### ゲーム管理
- ゲームオーバー / クリア制御
- 時間減速演出
- シーンリロード対応

### UI
- HPアイコン表示
- 状態に応じたUI切替

---

## 🎬 エフェクト

- 被弾時スプライト切り替え
- 死亡時エフェクト生成
- ボス撃破時エフェクト演出
- 時間スケール変化によるスローモーション演出

---

## 🛠 使用技術

- Unity
- C#
- Unity Input System
- Rigidbody2D
- Coroutine
- SceneManager
- SpriteRenderer
- （任意）Cinemachine

---

## 📦 セットアップ方法

1. リポジトリをクローン
2. Unity Hubでプロジェクトを開く
3. 必要パッケージをインストール
   - Input System
   - Cinemachine（使用する場合）
4. シーンを開いて再生

---

## ⚠ 注意点

- Missing Script が出る場合はスクリプトのコンパイルエラーを確認してください
- Cinemachine未導入の場合は関連コードを無効化してください
- Input System有効化が必要です
